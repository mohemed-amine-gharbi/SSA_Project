# Salp Swarm Algorithm (SSA) Project

This project demonstrates the **Salp Swarm Algorithm (SSA)**, a swarm-based evolutionary algorithm for optimization.

## Features
- Optimize any objective function (default: Sphere function)
- Interactive demo using Streamlit
- Adjustable parameters: dimension, number of salps, iterations

## Installation
```bash
git clone https://github.com/mohemed-amine-gharbi/SSA_Project
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
